## 🔄 如何使用这些文档

### 方案一：直接使用 Markdown
- 用 VS Code、Markdown 编辑器等直接查看
- 适合在线演示和版本控制

### 方案二：转换为 Word/PDF (推荐用于提交)

**安装 Pandoc**：
```bash
# macOS
brew install pandoc

# Windows: 从 https://pandoc.org 下载安装
# Linux: sudo apt-get install pandoc
```

**一键转换脚本**：
```bash
#!/bin/bash
cd docs/

# 转为 Word (.docx)
pandoc 软件需求规格说明书.md -o 软件需求规格说明书.docx
pandoc 系统设计说明书.md -o 系统设计说明书.docx
pandoc 测试报告.md -o 测试报告.docx

# 或转为 PDF
pandoc 软件需求规格说明书.md -o 软件需求规格说明书.pdf
pandoc 系统设计说明书.md -o 系统设计说明书.pdf
pandoc 测试报告.md -o 测试报告.pdf
```

3. 调整字体、行距、颜色
